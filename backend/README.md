# InterviewForgeAI Backend

Backend API for InterviewForgeAI.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Gemini API


## Modules

### Auth Module
- Register user
- Login user
- JWT authentication


### Resume Parser
- Upload resume
- Extract text
- Store resume data


### Interview Module
- Generate interview sessions
- Store questions
- Track answers


### Evaluation Module
- AI-based answer evaluation
- Generate feedback


## Running Backend

Install dependencies:

npm install


Start server:

npm run dev
